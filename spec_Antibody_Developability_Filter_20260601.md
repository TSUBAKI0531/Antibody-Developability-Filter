# Tool Spec: Antibody_Developability_Filter
**作成日**: 2026-06-01
**テーマ**: 深層生成モデルによる抗体設計
**対象期間**: 2026-06-01 〜 2026-06-01

## 背景
本ツールは、深層生成モデルによる抗体設計に関する研究において同定されたボトルネックである「生成AI（GAN、Diffusion、AbLM等）が生成した配列が、物理化学的安定性や高発現性（developability）に欠け、実際の製造や精製ラインで沈殿・凝集・低収量となるスクリーニング障壁を低減する。」を解決するために設計されました。

## 想定ユーザー
バイオ医薬品製造プロセス技術者、深層学習抗体設計者

## 解決する課題
生成AI（GAN、Diffusion、AbLM等）が生成した配列が、物理化学的安定性や高発現性（developability）に欠け、実際の製造や精製ラインで沈殿・凝集・低収量となるスクリーニング障壁を低減する。

## 入力データ
生成されたペア（VH-VL）抗体配列リスト（FASTA/CSV形式）
### データ形式
- VH/VL等のアミノ酸配列テキスト（FASTA形式、またはプレーンテキスト）
- CSV形式の配列リスト
### サンプル
```fasta
>Sample_VH
EVQLVESGGGLVQPGGSLRLSCAASGFTFSSYAMSWVRQAPGKGLEWVSAISGSGGSTYY
ADSVKGRFTISRDNSKNTLYLQMNSLRAEDTAVYYCAKDGYYGMDVWGQGTTVTVSS
```

## 処理フロー
1. ユーザーがStreamlit画面から配列を入力（ペーストまたはアップロード）
2. BioPython（ProteinAnalysis）を用いて、配列長、分子量、等電点、芳香族性（Aromaticity）等の物理化学的パラメータを解析
3. 統計モデル・類似度判定アルゴリズムにより結合特性およびオフターゲット／交差反応性スコアをシミュレート算出
4. 基準値超過の警告表示（エラー・赤色アラート）および結果の可視化グラフ描画

## 出力
各配列の製造適正予測スコア（High/Medium/Low）、沈殿/凝集リスクアノテーション、推薦配列のフィルタリング結果。

## 最低限の画面またはCLI機能
- テキストエリアによるアミノ酸配列ペースト入力
- 分析パラメータ（アラート閾値等）のスライダー制御
- 予測結合性・オフターゲット度のバーチャート可視化
- 判定結果（Pass/Alert/Fail）のカラーマーク表示

## 保存形式
予測結果およびパラメータのCSV（UTF-8エンコード）およびJSON形式での出力機能。

## エラー処理
- 不正アミノ酸文字（B, Z, X等）が含まれる場合の入力バリデーションチェックと警告メッセージ表示
- 配列長が極端に短い（CDR3領域が未検出など）場合のアラート処理

## 今後の拡張
- 機械学習モデル（GNN、トランスフォーマーモデル）のローカル推論エンジンのロードと高精度予測の統合
- 3D構造ドッキングシミュレーション結果（PDB）の取り込みおよび重ね合わせ表示機能

## 人手レビュー項目
- 予測閾値設定の生物学的妥当性の評価
- 他の主要な糖鎖/自己抗原データベースへの交差反応マッピング網羅性の確認
